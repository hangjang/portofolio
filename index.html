HTML

HTML의 기본 구조를 이해하고 웹 접근성을 고려해 다양한 태그를 사용하여 마크업을 작성 할 수 있습니다.

CSS

브라우저 호환성을 고려하여 CSS 코드를 작성 할 수 있고 CSS 애니메이션과 트랜지션을 구현 할 수 있으며 미디어쿼리를 활용해 반응형 레이아웃을 구현 할 수 있습니다.

SCSS

@mixin, @include를 활용해 코드의 재사용성을 높일 수 있습니다.

Swiper js

swiper js를 이해하고 적절하게 사용할 수 있습니다.

JavaScript

자바스크립트를 이해하고 동적으로 HTML 문서를 조작하고 다양한 종류의 이벤트(setInterval, setTimeout) 및 인터랙티브 기능을 구현 할 수 있습니다.

jQuery

jQuery를 이해하고 HTML 문서를 선택하고, 이벤트 핸들링과 클릭, 마우스 오버 및 터치 이벤트 등을 구현 할 수 있습니다.

React

React의 주요 개념인 컴포넌트와 state, props, Router를 이해하고 동적인 UI와 라우팅 방식을 구현 할 수 있습니다.

Vue.js

컴포넌트와 props를 이해했습니다.

AJAX

method:"GET" 으로 오픈 API 정보를 불러 올 수 있습니다.

JSON

html 내용을 txt 파일로 제작해 $.get 으로 불러올 수 있고 react에서 json 파일을 axios로 불러올 수 있습니다.

Photoshop

규격에 맞게 이미지를 잘라내고 색을 변경하고 이미지의 용량을 줄이는 등 웹 페이지 제작 시 필요한 이미지를 편집 할 수 있습니다.

Illustrator

기본적인 툴 사용이 가능하고 패스파인더 및 각종 도형 툴을 응용하여 로고 및 이미지를 제작 할 수 있습니다.

figma
